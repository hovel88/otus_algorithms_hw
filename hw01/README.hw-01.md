# ДЗ 01: Квадратные заклинания Гарри Поттера

## Цель

* Разгадать алгоритмы заклинаний по картинке

## Решение

Исходные картинки с заклинаниями расположены в каталоге `hw01/pics/`.  
Соответствующие исходники с алгоритмами находятся в файлах `hw01/src_*.cpp`.  
Для сборки исходников запустить `make`:

```bash
dmitry@lachugin:~/lachugin_algorithms_hw/hw01$ make
g++  -std=c++17 -Wall -I. -MD -O2 -o hw01_01 src_01.cpp
g++  -std=c++17 -Wall -I. -MD -O2 -o hw01_02 src_02.cpp
g++  -std=c++17 -Wall -I. -MD -O2 -o hw01_03 src_03.cpp
...
```
